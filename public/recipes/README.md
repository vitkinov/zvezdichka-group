# Рецепти - Markdown файлове

Тази папка съдържа рецептите като Markdown файлове.

## Формат на рецептите

Всеки файл с рецепта трябва да има следния формат:

```markdown
---
title: Заглавие на рецептата
mealType: breakfast
author: Име на автора
photo: https://example.com/image.jpg
---

Съдържание на рецептата...
```

## Типове ястия (mealType)

- `breakfast` - Закуска
- `lunch` - Обяд
- `dinner` - Вечеря
- `snack` - Междинно хранене
- `dessert` - Десерт

## Добавяне на нова рецепта

1. Създайте нов `.md` файл в тази папка с правилния формат (вижте примера по-долу)
2. Рецептата ще се покаже автоматично в приложението след рестартиране на приложението
3. Файловете се откриват автоматично - не е необходимо да се декларират в никакъв списък

## Пример

Вижте `example-recipe.md` за пример на правилно форматирана рецепта.
